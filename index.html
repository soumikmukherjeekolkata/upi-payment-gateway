Frequently Asked Questions

What platforms are supported by upi-intents?
The engine fully supports Android Chrome Intent URL routing (relying on target package matching) and iOS deep link URL schemes. Dynamic user-agent sniffing detects the client platform and switches link strategies instantly.
How does the QR Code Desktop Fallback work?
On desktop machines, clicking the payment buttons won't launch native apps. The playground generates a standard NPCI compliant QR Code representing the payment URI. Users can point any camera app or scanner from their UPI payment app to complete transactions safely.
Is this compliant with official NPCI regulations?
Yes. The library generates links matching the official National Payments Corporation of India (NPCI) UPI Linking Specifications v1.6. It performs parameter schema validations for VPA formats, amounts, notes, transaction identifiers, and transaction modes.
Can I download and integrate this library in my projects?
Absolutely! It is fully free for educational, personal, and developmental projects. Install it from npm registry using npm install upi-intents and check the modular source code in the /examples directory of the project.
